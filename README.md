# Astro-Guia — Chatbot de Exploração Espacial com IA

Astro-Guia é um chatbot interativo desenvolvido em Python utilizando o modelo Llama 3.3 70B através da API da Groq e integração com LangChain.

O projeto simula um especialista em exploração espacial capaz de responder perguntas técnicas e educativas sobre:
- astronomia;
- foguetes;
- missões espaciais;
- planetas;
- exploração do universo;
- tecnologias aeroespaciais.

---

# Funcionalidades

✅ Chat interativo via terminal  
✅ Memória contextual da conversa  
✅ Persona especializada em exploração espacial  
✅ Integração com API Groq  
✅ Uso do modelo Llama 3.3 70B  
✅ Tratamento básico de erros da API  

---

# Tecnologias Utilizadas

- Python 3
- LangChain
- Groq API
- Llama 3.3 70B
- Terminal/CLI
